Welcome to python-vibrato's documentation!
==========================================

`Vibrato`_ is a fast implementation of tokenization (or morphological analysis) based on the
Viterbi algorithm. This is a Python wrapper for Vibrato.

.. _Vibrato: https://github.com/daac-tools/vibrato
